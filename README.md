# austincv.com

## Yes, This Entire Website Was Vibe Coded

Every single line of HTML, CSS, and JavaScript on this site was written by Claude — Anthropic's AI — without me touching a code editor. Not once. Not even to fix a typo.

This was an experiment to see just how far you can push **Claude Code** (the CLI), Claude's automation features, and their mobile app before things go sideways. Spoiler: they didn't go sideways. It went suspiciously well.

## What "Vibe Coded" Means Here

I described what I wanted. Claude built it. I said "make the card flip look smoother." Claude made it smoother. I said "add a mobile swipe thing." Claude added a mobile swipe thing. I contributed vibes. Claude contributed the actual work.

## The Automation Rabbit Hole

Part of the experiment was exploring the full surface area of what Claude can do:

- **Claude Code CLI** — writing and editing files, running commands, managing git branches, the whole lot
- **Claude on mobile** — prompting on the go, because apparently you don't need a laptop to ship a website anymore
- **Agentic workflows** — Claude planning multi-step tasks, making decisions, and executing them without hand-holding

## About This README

This README? Also written via a prompt. Specifically, I was on the **London Underground** on my way to a friend's house **to play board games** when I typed "add a small readme, make it funny and cheerful" into the Claude mobile app.

The tube had signal for approximately 40 seconds. It was enough.

## What the Site Actually Is

A personal CV site for Austin Chungath Vincent. It has:
- An ID card that flips as you scroll (desktop)
- A swipeable card carousel (mobile)
- Six roles, one card each
- Zero frameworks, zero build steps, zero regrets

## Lessons Learned

1. Claude Code is genuinely capable of building real things end-to-end
2. You can have a fully functioning personal website without writing any code yourself
3. The tube is a perfectly valid development environment
4. Board games are better when you've already shipped something on the way there
